# joyn

## Install

```shell
brew install nicholasdower/tap/joyn
```

## Uninstall

```shell
brew uninstall joyn
```

## Help

```
usage: joyn [-d <delimiter>] [file ...]

Join lines

Options

    -d, --delimiter  The line delimiter.
    -h, --help       Print help.
    -v, --version    Print version.
```
