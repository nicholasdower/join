# join

## Install

```shell
brew install nicholasdower/tap/join
```

## Uninstall

```shell
brew uninstall join
```

## Help

```
usage: join [-d <delimiter>] [file ...]

Description

    Join lines, optionally using the specified delimeter.

Options

    -h, --help       Print help.
    -v, --version    Print version.
```
